Approximation by Rationals

2025–2026

Year 4 Group Project · University of Edinburgh (Supervisor: Professor Jim Wright)

This is my final-year pure mathematics project on Diophantine approximation. The project is centred on a fundamental question: how well can real numbers be approximated by rationals? Building from that question, the report develops a structured account of continued fractions, Liouville’s theorem, Thue’s theorem, and their connections with Pell equations, binary homogeneous polynomials, and Dirichlet’s Unit Theorem. The project is primarily theoretical and expository in nature: it requires not only understanding individual results, but also organising them into a coherent mathematical narrative with clear logical progression. Among my projects, this one best represents my long-term interest in pure mathematics, especially number theory and adjacent algebraic topics.

Constraint Programming for Mathematics Timetabling

2026

TAOR Final Project · University of Edinburgh · Team Leader

This project investigates the timetable feasibility of a proposed redesign of the honours Mathematics curriculum at the University of Edinburgh. Using Google OR-Tools, I helped develop a large-scale CP-SAT model that jointly captures course scheduling, student choice structures, fixed outside-school timetables for joint-degree students, and a range of timetable-quality rules. A key modelling idea is student-type aggregation: instead of modelling every student individually, students with the same programme structure and external timetable are grouped together to reduce model size while preserving the essential combinatorial structure of the problem. We then designed and compared multiple scenario variants to analyse how lunch-break rules, 9am restrictions, compactness requirements, and consecutive-lecture constraints affect feasibility. The final results show that, under an appropriate rule set, a zero-conflict timetable can be achieved at scale, while overly restrictive lecture-block rules disproportionately harm joint-programme students. This project is one of the strongest examples of my work in optimisation, mathematical modelling, and large-scale computational experimentation.

Random-Walk-Based Graph Neural Networks

Summer 2025

Vacation Research Project

This summer research project focuses on expressivity in graph neural networks. Starting from two major limitations of standard message-passing GNNs—oversmoothing and oversquashing—the project explores whether random-walk-based approaches can provide richer representations and more flexible graph modelling. My work includes reviewing the mechanisms and limitations of standard GNN architectures such as graph convolutional and graph attention networks, studying existing random-walk-based methods, and investigating theoretically motivated random walk schemes for improved performance. The project also extends the discussion to hypergraphs, where higher-order relationships cannot be naturally captured by ordinary graph structures alone. I see this as a particularly representative project because it connects graph theory, machine learning, representation learning, and theoretical analysis in a way that matches my broader interest in mathematically oriented machine learning.

Quantitative Modelling of Winter Peak Electricity Demand in Great Britain

Apr–May 2025

Statistical Modelling Project · Team Leader

In this project, I led a team in building a statistical model for winter peak electricity demand in Great Britain using historical data from 1991 to 2014. The model uses a multi-factor linear regression framework that incorporates renewable generation, seasonal effects, and weekday structure in order to improve both interpretability and predictive performance, with the broader goal of estimating reserve-capacity needs under severe winter conditions. Beyond the regression model itself, I also built an automated model-selection workflow based on five-fold cross-validation together with AIC/BIC comparisons, which made validation more systematic and significantly improved efficiency. This project is well suited for a personal website because it shows that I can work with real data, design interpretable statistical models, and integrate modelling, validation, and analysis into one coherent workflow.

BoxCar Ride-Sharing Simulation Study

Feb 2026

Data-Driven Discrete-Event Simulation Project · Team Leader

This project studies the operational performance of a ride-sharing system through a data-driven discrete-event simulation. Together with a teammate, I built a model that captures rider arrivals, driver availability, distance-based matching, trip execution, and the associated revenue and cost structure. We first examined whether the parameter values given in the project brief were statistically consistent with the observed data, and found that the default configuration drives the system close to saturation, producing longer waiting times, higher cancellation rates, and weaker net revenue. We then re-estimated the key parameters using maximum likelihood methods and compared the adjusted baseline with the original setting. Finally, we designed several scenarios to analyse trade-offs between rider service quality, driver earnings, profitability, and system balance. A central conclusion of the project is that simply oversupplying drivers is not the most robust way to improve service; better matching efficiency and reduced empty miles are often more sustainable operational improvements.

Strategic Workforce Allocation Optimization

Apr 2025

Optimisation Modelling Project · Team Leader

This project considers a multi-project workforce allocation problem for a consulting firm. The goal is to allocate consultants across projects, locations, and consultant levels while respecting role restrictions, travel costs, bench policies, and other business rules, and at the same time maximise total profit. I used Xpress to formulate and solve a linear/integer optimisation model for 13 projects across multiple regions, and then extended the analysis to more realistic strategic scenarios involving demand variability, remote-work penalties, outsourcing costs, and Irish transport disruption. The results show, for example, that moderate use of remote work can significantly reduce travel costs, while the marginal gains flatten once remote capacity reaches roughly 15% of project demand. The project also highlights how heavy dependence on cross-regional deployment weakens resilience under disruption. Overall, this is a good applied-optimisation project that demonstrates how mathematical programming can support real managerial decisions.

Multi-Stage Laptop Manufacturing System Simulation Study

Apr 2026

Manufacturing System Simulation Project · Team Leader

This project develops a Simul8 discrete-event simulation of a multi-stage laptop manufacturing system. We first carried out input analysis on interarrival times and multiple processing-time datasets, using AIC, BIC, KS testing, and maximum likelihood estimation to select appropriate probability models. We then embedded mechanisms such as blocking, rework, inventory control, component matching, and order-level synchronisation into a full simulation framework. The baseline model revealed poor system performance, with an average cycle time of about 1575 minutes and the case-manufacturing line identified as the main bottleneck. Based on these findings, we designed a sequence of progressive improvement experiments that exposed layered bottlenecks in aluminium supply, hard-disk availability, and workbench capacity. The final improved model reduced average cycle time to about 1242 minutes, increased throughput from 2.82 to 7.14 laptops per hour, and significantly lowered the rejection rate. This project is a strong example of my work in input modelling, system diagnosis, experimental simulation, and operational interpretation.
